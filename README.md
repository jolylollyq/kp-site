# KP Site

Коммерческое предложение на разработку сайта для ветеринарной клиники.

## Технологии

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion

## Разработка

```bash
npm install
npm run dev
```

## Деплой на GitHub Pages

```bash
npm run export
```

Проект автоматически деплоится на GitHub Pages при пуше в main.
